bash基本命令介绍

bash命令的基本语法

命令名称 [-选项] [参数] (方括号包起来的都是 可选的)

##　cd
全称: change director
功能: 改变当前所在的目录

cd 文件夹的路径

~: 指的其实是当前用户的用户根目录
等价于:/c/Users/Ryan/Desktop/

..: 指上一级目录

pwd

全称: Print Work Directory
功能: 打印当前所在的目录

ls

全称: list
功能: 展示当前路径下的所有文件

有参数的情况
功能: 不切换文件夹 , 就直接展示指定的文件夹下的所有的文件信息

用法: ls 文件夹路径

有选项的情况
-a选项的意思是: all 也就是将所有内容全部展示出来 , 包括隐藏文件
-l选项的意思是: list 将所有的内容的详细信息以表格的形式显示出来

用法: ls -a
ls -l

touch

功能: 创建文件
用法: 在当前目录下创建文件 touch 文件名
在指定目录下创建文件 touch 路径/文件名

mkdir

功能: 创建文件夹
用法: touch 文件夹
touch 路径/文件夹

rm

remove
功能: 删除文件
注意: 这个删除不会进回收站
用法: rm 文件名
rm 目录/文件名

rm命令也可以用来删除文件夹 , 但是需要加上 -r
即: rm -r 文件夹名称
强制删除:
rm -f

rmdir (几乎用不到)

功能: 删除非空文件夹

cp

copy
功能: 复制文件 或者文件夹

cp 要复制的文件 复制到哪里去

复制文件夹: cp -r 要复制的文件夹 复制到哪里去

mv

move
移动文件 文件夹
用法: mv 要移动的文件 要移动到哪里去
‘常被用作重命名操作’

清空屏幕

clear
reset

查看文件内容

cat 文件名
cat 路径/文件名

less 可以将文件内容分页展示
下一页: 空格 回车 或者 鼠标滚轮 或者 方向键

退出内容展示: 按q键

echo

将指定内容输出到屏幕上

重定向

‘>’: 覆盖操作
‘>>’: 追加
